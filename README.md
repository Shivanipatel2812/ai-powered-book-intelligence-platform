# 📚 AI-Powered Book Intelligence Platform

## 🚀 Project Overview
This is a full-stack web application that allows users to manage books and ask AI-based questions about them.

---

## ✨ Features
- Add, update, delete books
- Search & filter books
- Book detail page
- Ask questions about books (AI Q&A)
- REST API backend

---

## 🛠 Tech Stack
- Frontend: React (Vite)
- Backend: Django REST Framework
- Database: SQLite
